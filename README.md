# Shadows of the Ancient Forest: Web Board

A specialized mystery-solving interface for the "Shadows of the Ancient Forest" investigation.

## Features
- **Investigator Dossiers**: Detailed stats and backgrounds for the active team.
- **Clue Repository**: Persistent tracking of discovered items and artifacts.
- **Suspect Analysis**: Deep-dive into motives and lore of the forest's threat.
- **Top-Aligned Interface**: Optimized for quickly switching between clues without excessive scrolling.

## Tech Stack
- React 19
- Vite 8
- Vanilla CSS

## Local Development
```bash
npm install
npm run dev
```

## Deployment
Deployed via Vercel for real-time collaborative investigation updates.
